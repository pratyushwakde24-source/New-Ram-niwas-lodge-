# New Ram Niwas Lodging House - Dark Luxury Premium Hospitality Website

A sophisticated, ultra-premium city lodging website designed for **New Ram Niwas Lodging House**, located at 11 New Nagardas Road, Andheri East, Mumbai.

---

## 🎨 1. Dark Luxury Design System & Aesthetic

Inspired by **Apple**, **The Oberoi**, **Airbnb**, and **Booking.com Premium**, this design elevates a city lodging hotel into an elegant, trustworthy, and modern brand without redesigning it into an unrealistic 5-star resort:
* **Color Palette:** Ultra-dark background (`#090909`), sleek secondary cards (`#161616`), and refined **Warm Gold** accents (`#d4af37`).
* **Typography:** Pairing Google Fonts **Playfair Display** for elegant editorial headers with clean **Inter** for professional UI copy.
* **UI Details:** Soft glassmorphism (`rgba(22, 22, 22, 0.82)` with backdrop blur), subtle linear/radial gradients, rounded corners, and butter-smooth GSAP scroll reveal animations.

## 📸 How to Replace Placeholders with Your Genuine Photos

To update the website with your real hotel photography from Justdial or smartphone captures:
1. Create an `/images/` folder inside this project directory.
2. Save your real `.jpg` or `.png` photos (e.g., `hero-real-exterior.jpg`, `standard-ac-room.jpg`, `reception-desk.jpg`, `bathroom-clean.jpg`).
3. Open `index.html` in any text editor and replace the placeholder blocks inside `.room-img-container` or `.masonry-item` with standard HTML image tags:
   ```html
   <img src="images/standard-ac-room.jpg" alt="Standard AC Room at New Ram Niwas Lodging House" style="width: 100%; height: 100%; object-fit: cover;">
   ```

## ⚡ 4. Technical Performance & SEO Implementation

* **GSAP & ScrollTrigger Animations:** Integrated via CDN with custom `.gsap-reveal` classes for elegant fade-in and scale-up scroll reveal effects without excessive flashiness.
* **Lighthouse Optimized:** Preloads fonts and hero images, utilizes deferred JavaScript execution (`defer`), explicit container sizes for CLS optimization, and semantic ARIA accessibility labels.
* **SEO Suite:** Features Open Graph tags, Twitter Cards, canonical link tags, valid **`sitemap.xml`**, **`robots.txt`**, and official **Schema.org `Hotel` JSON-LD** structured data for top Google local search rankings.
* **Direct WhatsApp Booking Form:** Compiles stay dates, guest names, room preference, and party size into a cleanly formatted inquiry string and launches WhatsApp directly to `+91 9820403685`.
